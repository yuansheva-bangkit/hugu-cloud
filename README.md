### Getting Started

HugU Backend requires [Node.js](https://nodejs.org/) v10+ to run.
Clone the repositories, Install the devDependencies and start the server.

1. `git clone https://github.com/yuansheva-bangkit/hugu-cloud.git`
2. `cd hugu-cloud`
3. `npm i`
4. `npm run start-dev`
